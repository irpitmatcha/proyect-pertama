Tentang Proyek

Repo ini sekarang punya arah yang lebih jelas.

Tujuan awal

Membangun proyek GitHub sederhana yang tidak berhenti di file placeholder, tetapi langsung punya tampilan presentable untuk portofolio atau eksperimen pribadi.

Arah pengembangan

Kamu bisa lanjut menambahkan gameplay, skor, collision, atau memindahkannya ke struktur JavaScript yang lebih modular saat ide game-nya sudah matang.

Isi Repo

Fondasi yang pas untuk mulai membangun.

01

Visual lebih kuat

Palet warna hangat, tipografi tegas, dan layout yang tidak terasa generik.

02

Responsif

Komponen menyesuaikan ukuran layar tanpa perlu framework tambahan.

03

Interaksi ringan

Tombol tips acak memberi sentuhan JavaScript tanpa membuat struktur jadi rumit.

Langkah Berikutnya

Roadmap singkat untuk mengubah ini jadi game sungguhan.

1

Tentukan aturan game block: objektif, input pemain, dan kondisi kalah/menang.

2

Pisahkan logika game ke file JavaScript yang lebih terstruktur.

3

Publikasikan ke GitHub Pages agar hasilnya bisa dibuka lewat link publik.
